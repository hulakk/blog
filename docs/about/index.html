只构建

jekyll build

git快捷提交（使用日期作为日志）

Windows

CMD： 使用环境变量%date%和%time%

git commit -m "%date% %time%"

Powershell: 使用date命令的输出

$d=date
git commit -m "$d"

Linux

使用date命令的输出

git commit -m "$(date)"
